# blog.comame.xyz

[https://blog.comame.xyz](https://blog.comame.xyz)

Next.js + Vercel

## インストール

1. entries.json に空配列を入れる。`$ echo [] > entries/entries.json`
1. .env に設定を入れる
1. `npm run compose` を実行し、タイトル、ファイル名を入力する。
