Gmail のフォローアップ機能を停止する

2018-07-27
source history
  • Google

新デザインの Gmail には、返信し忘れたメールや対応が必要なメールを判別し、受信トレイの先頭に表示してくれる機能が追加されました。
このフォローアップ機能は便利ですが、一方で不要な方もいるでしょう。この機能を停止する方法を紹介します。

モバイル

  1. Gmail アプリを開く
  2. 左上のメニューアイコンをタップして、サイドバーを開く
  3. 最下部の「設定」を開き、フォローアップ機能を停止したいアカウントを選択する
  4. 「アクションの提案」セクション内の「返信とフォローアップ」を開く
  5. 停止したいもののチェックボックスを外す

デスクトップ

  1. Gmail を開く
  2. 右上の歯車アイコンを押し、設定を開く
  3. 「概要」タブの「アクションの提案」セクションに移動する
  4. チェックボックスを外す

有効にする手順もこれと同様です。
