Windows 10 の ssh で Permission refusedが出る問題

2020-02-19
  • troubleshoot
  • Windows

対処法

ssh-agent を有効にして、ssh-add する。

Set-Service -Name ssh-agent -StartupType Manual
Start-Service ssh-agent

ssh-add ~/.ssh/id_rsa
